# 📝 To-Do App

## 📌 Overview

The **To-Do App** is a simple and interactive task management application developed using **HTML, CSS, and JavaScript**. It helps users organize their daily tasks efficiently with a clean user interface, progress tracking, motivational quotes, and task persistence using Local Storage.

---

## ✨ Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Progress bar with completed task counter
* Motivational quote changes after completing all tasks
* Confetti celebration when every task is completed
* Local Storage support (tasks remain after refreshing the page)
* Responsive Glassmorphism UI
* Beautiful motivational background

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API

## 💻 How It Works

* Users can add, edit, and delete tasks.
* Completed tasks are marked with a checkbox and become read-only.
* The progress bar updates automatically based on completed tasks.
* When all tasks are completed:

  * 🎉 Confetti animation is displayed.
  * 💬 A motivational success message replaces the default quote.
* All tasks are automatically saved in the browser using **Local Storage**, allowing data to persist even after refreshing the page.
